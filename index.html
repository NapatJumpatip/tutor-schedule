เชื่อมต่อ Google Calendar

ทำครั้งเดียวแล้วใช้ได้ตลอดค่ะ ตาม 3 ขั้นตอนนี้:

1
ไปที่ Google Cloud Console → สร้าง Project ใหม่ → เปิดใช้งาน Google Calendar API
2
ไปที่ APIs & Services → Credentials → สร้าง OAuth 2.0 Client ID เลือก "Web application" → ใส่ http://localhost และ file:// ใน Authorized origins
3
คัดลอก Client ID แล้วใส่ด้านล่างค่ะ

Client ID เก็บไว้ในเบราว์เซอร์ของหยกเท่านั้น ไม่ได้ส่งไปที่ไหนค่ะ
